  МЕНИДЖЪР НА ЧУВСТВАТА — Инструкции за качване
═══════════════════════════════════════════════════════

Ще качиш 4 файла в Railway.app — безплатна платформа.
След това ще получиш URL, който работи от всяко устройство.

ФАЙЛОВЕ (всичките 4 трябва да са заедно):
  ├── server.py
  ├── index.html
  ├── requirements.txt
  └── Procfile

───────────────────────────────────────────────────────
СТЪПКА 1 — Създай безплатен акаунт в GitHub
───────────────────────────────────────────────────────
1. Отвори https://github.com
2. Натисни "Sign up"
3. Въведи email, парола, потребителско име
4. Потвърди email-а

───────────────────────────────────────────────────────
СТЪПКА 2 — Качи файловете в GitHub
───────────────────────────────────────────────────────
1. Влез в GitHub
2. Натисни зеленото бутче "New" (горе вляво)
3. Repository name: emotii
4. Избери "Private" (личен)
5. Натисни "Create repository"
6. На следващата страница натисни "uploading an existing file"
7. Провлачи (drag & drop) всичките 4 файла
8. Натисни "Commit changes"

───────────────────────────────────────────────────────
СТЪПКА 3 — Качи в Railway
───────────────────────────────────────────────────────
1. Отвори https://railway.app
2. Натисни "Start a New Project"
3. Избери "Deploy from GitHub repo"
4. Влез с GitHub акаунта си (Authorize Railway)
5. Избери хранилището "emotii"
6. Railway автоматично ще открие Python и ще стартира сървъра

───────────────────────────────────────────────────────
СТЪПКА 4 — Вземи твоя URL
───────────────────────────────────────────────────────
1. В Railway → твоят проект → таб "Settings"
2. Намери "Public Networking" → "Generate Domain"
3. Ще получиш URL от вида: https://emotii-xxxx.up.railway.app
4. ТОВА е твоят личен адрес — отваря се от всяко устройство

───────────────────────────────────────────────────────
ГОТОВО
───────────────────────────────────────────────────────
Отвори URL-а от Mac, телефон, таблет — данните се синхронизират
автоматично защото се пазят в базата данни на сървъра.

Безплатният план на Railway включва $5 кредит/месец,
което е достатъчно за лична употреба.

При въпроси — попитай в чата.
